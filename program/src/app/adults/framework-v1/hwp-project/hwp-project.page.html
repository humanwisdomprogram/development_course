The HumanWisdom App

The HumanWisdom app is a life-companion and a step by step guide to help users understand themselves and live with wisdom. This is not based on any ideology, but on looking and learning about ourselves. It has 59 modules that can help you navigate all of life’s challenges with calm. New content is added regularly and there are many features to explore.

Help us Improve

We believe the app belongs to all its users, and everyone is welcome to contribute to help us improve it. It could be an insight into how the mind works, or a better way of presenting this information. We have a shared humanity and are just exploring the mind we all share. There is no authority. Nothing is true unless you see it clearly yourself. This understanding brings illumination, which can lead to transformation. If you have a comment that can help us improve the app please email comment@humanwisdom.me
